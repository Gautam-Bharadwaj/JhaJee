## Getting Started

To run this project locally, follow these steps:

### Prerequisites

You'll need Git and optionally Node.js installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Gautam-Bharadwaj/JhaJee.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd JhaJee
   ```

3. **Install dependencies (if applicable):**
   ```bash
   npm install
   ```

4. **Start the development server (if using a bundler/framework):**
   ```bash
   npm run dev
   ```
   *Note: If this is a vanilla HTML/CSS/JS project, you can simply use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code to run the `index.html` file.*
